# wishlist

Wish list is a project that manages customers and their favorite products.

## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

You will need [Java][] >= 1.8

[java]: https://www.oracle.com/java/technologies/javase-jdk8-downloads.html

## Make Commands

### dependencies:
    Install dependencies

### run-dev:
    Run server

### repl:
    Run repl (Read-Eval-Print Loop)

### build-docker:
    Build docker image

### remove-image-docker:
    Remove docker image

### docker-run:
    Run app using Docker

### docker-stop:
    Stop docker app

### tests:
    Run all tests

### lint:
    Run check the formatting of your source files

### lint-fix:
    Format code

### build-package:
    Build jar package
